# GIF Generator (Python)

A simple Python project that generates an animated GIF from multiple image frames using the ImageIO library.

---

## 📌 Project Overview

This project demonstrates how to combine several image frames into a single animated GIF using Python.
The program reads image files from a folder and merges them into a looping GIF animation.

---

## ⚙️ Technologies Used

* Python
* ImageIO

---

## 📂 Project Structure

```
gif-project/
│
├── Frames/
│ ├── Pixel art hero 1.png
│ └── Pixel art hero 2.png
│
├── Create_gif.py
└── Pixel_Hero.gif
```

## ▶️ How to Run the Project

1. Install the required library:

pip install imageio

2. Run the Python script:

python Create_gif.py

---

The script generates an animated GIF from the image frames.
